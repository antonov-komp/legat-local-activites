# Legat. Контакты компании

Кастомное activity для бизнес-процессов Битрикс24.

- вход: **УНП**;
- запрос: `GET https://api.legat.by/api2/by/contacts`;
- выходы: статус, телефоны, факс, email, сайты, JSON `contacts`.

Ключ API: `local/legat_api_key.php` (`LEGAT_BY_DATA_API_KEY`) или `Option main.legat_by_data_api_key`.
