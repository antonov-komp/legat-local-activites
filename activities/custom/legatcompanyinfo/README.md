# Legat. Общие сведения о компании

Кастомное activity для бизнес-процессов Битрикс24.

Назначение:
- принимает `УНП`;
- выполняет запрос к `GET https://api.legat.by/api2/by/data`;
- возвращает данные **отдельными полями** в блоке доп. результатов (общие сведения, налоговая инспекция, статусы, ФСЗН, суды, закупки, финансы и т.д.);
- поле **«Статус запроса»** — краткий итог выполнения (успех или текст ошибки).

Ключ API (один из вариантов):
- в `local/php_interface/init.php`: `define('LEGAT_BY_DATA_API_KEY', 'ваш_ключ');`
- либо опция `main` → `legat_by_data_api_key` (задаётся через `Option::set` или инструментами администратора).

В репозиторий ключ не коммитить.
